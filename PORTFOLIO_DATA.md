## Design Notes (Current Portfolio)

- **Style:** Minimal, bold typography (black & white), editorial feel
- **Font style:** Heavy all-caps headings (font-black, tracking-tight)
- **Color palette:** Black, white, gray tones — no bright accent colors
- **Animations:** Scroll-triggered fade/slide-in via Framer Motion; parallax on Hero
- **Layout:** Max width 7xl, responsive grid (single col mobile → 2-col desktop)
- **Image treatment:** Grayscale photo that colorizes on hover (About section)
- **Project cards:** List rows (info left, thumbnail + Live/APK links right), hover highlight; Featured badge; APK download link shown for projects shipped as Android apps (Arena, Spendly)
- **Sections:** Hero → About → Skills → Projects → Experience → Contact → Footer
